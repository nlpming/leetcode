### 343.整数拆分
- 状态转移方程：`dp[i] = max{(dp[i-j*j], (i-j)*j), j=1,2,...,i-1}`

### 91.解码方法
- 1.dp[i]表示字符s[0...i]的解码方法；答案返回dp[n-1]；
- 2.如果当前字符 `s[i] != '0'`，则表示s[i]可以单独解码；则 `dp[i] += dp[i-1]`；
- 3.如果`s[i-1] != '0' && s[i-1,i] <=26`，表示s[i-1,i]可以单独解码；则 `dp[i] += dp[i-2]`；
- 4.注意对于`i=0, i=1`的特殊处理；
